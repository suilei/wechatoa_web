什么是行政法
行政许可法 即中华人民共和国行政许可法。 中华人民共和国主席令第七号，由中华人民共和国第十届全国人民代表大会常务委员会第四次会议于2003年8月27日通过，现予公布，自2004年7月1日起施行。为了规范行政许可的设定和实施，保护公民、法人和其他组织的合法权益，维护公共利益和社会秩序，保障和监督行政机关有效实施行政管理，根据宪法规定制定本法。全文共八章八十三条。
